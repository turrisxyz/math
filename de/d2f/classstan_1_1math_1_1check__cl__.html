stan::math::check_cl_< T > Class Template Reference

Detailed Description

template<typename T>
class stan::math::check_cl_< T >

Represents a check in kernel generator expressions.

When a check is assigned an expression it will throw if any value in the result of the expression is false.

Template Parameters
Ttype of matrix_cl or expression that is checked
